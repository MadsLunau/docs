The API lets you run your Customer Agent outside a ticketing system: build conversational AI features directly into your platform, or use it internally. Companies like Cursor and Lovable use it today.

## How it works

Tickets come in through the API instead of a ticketing system. Your Customer Agent handles them with the same configurations, tools, and guardrails, and you control the experience around it.

## Reference

The full API reference lives at [app.parahelp.com/api/docs](https://app.parahelp.com/api/docs).
