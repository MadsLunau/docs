When you approve a configuration change in a Manager Agent task, such as adding a new searchable web page, change to a memory file, or a new tool, it appears on the "Releases" page.

## What's in a release

Each release is a versioned set of published changes: v1, v2, v3, and so on.<br /><br />Open a release to see exactly what changed, line by line, for example "Updated base instructions" or "Created Trial Refund tool".
