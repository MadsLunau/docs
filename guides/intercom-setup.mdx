## Create Parahelp team member

1. Open the **Settings** page and within the **Workspace** tab, select [Teammates](https://app.intercom.com/a/apps/_/settings/workspace/teammates?tab=invited):
   {/* SCREENSHOT: Settings > Workspace > Teammates */}
2. Click the \+ **Add new** button and then **\+ Add new teammates**:
   {/* SCREENSHOT: Add new teammates */}
3. Enter the email address (\[company\]@parahelp.com) and click **Continue**.
4. Set the **following permissions**:
   - "Seat type" → "Full seat"
   - "Conversation access" → "All conversation"
   - "Settings" → "Can manage teams"
   - "Data and security" → All permissions\*
   - "Apps and Integrations" → All permissions
   - "Knowledge" → All permissions
   - "Automation" → All permissions
   - "Inbox" → All permissions
   - "Reports" → All permissions
   \*These permissions allows us to export macro usage data to incorporate your most frequently used macros into Parahelp's knowledge base and it also allows the AI agent to transfer tickets to team members.
