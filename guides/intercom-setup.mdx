---
title: "Intercom connection setup"
description: "Create an Intercom developer app to connect Parahelp to your workspace. Takes about 5 minutes."
---

<Steps>
  <Step title="Create the developer app in Intercom">
    Go to **Settings → Developer Hub**, click **New app**, and name it Parahelp. 

    <Frame>
      ![Image](/images/image-20.png)
    </Frame>

    <Frame>
      ![Image](/images/image-21.png)
    </Frame>

    <Frame>
      ![Image](/images/image-22.png)
    </Frame>

    Open the app you just created.

    {/* SCREENSHOT: Intercom Developer Hub, New app */}
  </Step>
  <Step title="Set the API version">
    In the app settings, open **API version**, set it to **2.14**, and click Save.

    <Frame>
      ![Image](/images/image-20.png)
    </Frame>

    {/* SCREENSHOT: API version setting */}
  </Step>
  <Step title="Configure webhooks">
    In the app settings, open **Webhooks** and set the Endpoint URL to `WEBHOOK_ENDPOINT_URL`. 

    <Frame>
      ![Image](/images/image-22.png)
    </Frame>

    Then add these topics:

    - conversation.admin.assigned
    - conversation.admin.closed
    - conversation\_part.tag.created
    - conversation.admin.opened
    - conversation.admin.replied
    - conversation.admin.snoozed
    - conversation.admin.unsnoozed
    - conversation.user.created
    - conversation.user.replied

    <Frame>
      ![Image](/images/image-23.png)
    </Frame>

    {/* SCREENSHOT: Webhooks configuration */}
  </Step>
  <Step title="Set permissions">
    Under **Authentication**, enable the following permissions, then click Save:

    - Read one user and one company
    - Read conversations
    - Write conversations
    - Read tags
    - Write tags
    - Write tickets
    - Read tickets
    - Read admins
    - Update admins
    - Read and list users and companies
    - Write users and companies
    - Read and list articles
    - Read and write articles
    - Read and write users
    - Write data attributes
    - Read data connectors

    {/* SCREENSHOT: Authentication permissions */}
  </Step>
  <Step title="Add your access token in Parahelp">
    In the Intercom app, open **Configure → Authentication** and copy the **Access token**, then open **Basic information** and copy the **Client secret**.

    In Parahelp's onboarding task, paste each value into the secure form when prompted. Parahelp validates them, automatically detects your workspace and region, and looks up your teammates so you can confirm which one acts as the Customer Agent.

    {/* SCREENSHOT: Parahelp secure form */}
  </Step>
</Steps>

Once connected, [assign conversations to Parahelp](/guides/intercom-assigning).
