## Custom connections

### Capabilities

- **MCP:** Connect any MCP server to give the Internal Agent and Customer Agent access to the tools that server exposes, such as a custom CRM, a data warehouse, or internal scripts.
- **API:** Connect a custom API endpoint, such as your own backend or a third-party platform like HubSpot. The Internal Agent will build a custom tool that wraps the endpoint and test it against real data before going live. For example, a "look up this customer in our internal admin panel" tool.

### Setup

Prompt the Internal Agent. Tell it what you'd like to connect and what you want the Customer Agent or Internal Agent to be able to do with it. The Internal Agent will guide you through wiring it up, requesting any secrets it needs, and creating the matching tool or script.
